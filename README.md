# Pathum Chamara Portfolio

Modern React + Tailwind CSS portfolio with Framer Motion animations.

## What was improved

- Changed font system to a premium combo: **Sora** for headings and **Outfit** for body text.
- Improved dark theme, gradient colors, glassmorphism cards, hover states, and button glow.
- Added a custom favicon.
- Added `src/assets/README.md` with image replacement instructions.
- Fixed a small invalid React prop in the Services component.
- Verified production build successfully.

## Run the project

```bash
npm install
npm run dev
```

## Build the project

```bash
npm run build
```

## Add your photos

Add your profile photo and project images inside `src/assets/`:

- `profile.jpg`
- `project1.jpg`
- `project2.jpg`
- `project3.jpg`
- `project4.jpg`

Then update/uncomment the image code inside the relevant component files.
